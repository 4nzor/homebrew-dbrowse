# homebrew-dbrowse

Homebrew tap for [dbrowse](https://github.com/4nzor/dbrowse) - terminal database browser.

## Installation

```bash
brew tap 4nzor/dbrowse
brew install dbrowse
```

## Usage

```bash
dbrowse
# or
dbrowser
```

## Updating

```bash
brew upgrade dbrowse
```

Or use built-in update checker:
```bash
dbrowse --update
```

## Features

- Multi-database support (PostgreSQL, MySQL, SQLite, MongoDB, ClickHouse)
- Interactive TUI with mouse and keyboard navigation
- Data filtering and sorting
- CSV/JSON export
- Connection management
- Automatic update checker
